# 🏦 Banking System (C++)

A console-based Banking System developed in C++ using Object-Oriented Programming and File Handling. This project simulates basic ATM functionalities such as account creation, login, deposit, withdrawal, and transaction history.

---

## 📌 Project Overview

This system allows users to create an account, securely log in, and perform banking operations. All account data is stored using file handling, ensuring data persistence.

---

## 🚀 Features

- 🔐 Account Creation (User ID & Password)
- 🔑 Secure Login System
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Balance Inquiry
- 📜 Transaction History with Date & Time
- 💾 File Handling for Data Storage

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)

- File Handling (ifstream, ofstream)
- Arrays and Structures

## ▶️ How to Run

1. Open the code in any C++ compiler (Dev C++, VS Code, etc.)
2. Compile the program
3. Run the executable
4. Choose:
   - Create Account (c)
   - Login (l)
5. Perform operations like deposit, withdrawal, and balance check
## 📊 Sample Functionalities

- Create a new account with ID and password  
- Login using credentials  
- Deposit or withdraw money  
- View current balance  
- Check transaction history with timestamps
